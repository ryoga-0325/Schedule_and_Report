社内スケジュール・日報管理

概要
・IDとパスワードでログインする
・全社員のスケジュールがカレンダー形式で確認できる
・ログインしたユーザーはカレンダーからの日別スケジュールと日報を書き込める

node.js, express 
SQL Server EXPRESS(Tedious)

画面
1．ログイン画面
	・IDとパスワードを入力してログインする
	・認証にはpassportを使用（PWの暗号化は未実装）
2. 当日スケジュール一覧画面
	・ログイン成功時に遷移する画面
	・当日の全員のスケジュールを一覧表示する（ログインしているユーザー本人はハイライト表示）
3. カレンダー画面
	・ログインしているユーザーの当月のスケジュールをカレンダー形式で表示する
	・カレンダーは前月・次月の移動、年月を選択して切り替え可
	・社員選択ボックスで選択した別の社員のスケジュールも表示できる
	・日付を押下で当日のスケジュール一覧画面、スケジュール内容を押下で登録・編集画面へ遷移する
4. スケジュール一覧画面
	・カレンダー上の日付を押下すると全員のスケジュールを一覧表示する（ログインしているユーザー本人はハイライト表示）
5. スケジュール・日報　登録・編集画面
	・スケジュールと日報を登録・編集する
	・送信ボタン押下で登録し、カレンダー画面に戻る
	・ログインしているユーザー以外の社員のスケジュールを選択していた場合は、登録と編集はできない
